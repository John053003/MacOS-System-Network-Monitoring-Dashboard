# MacOS System & Network Dashboard

A real-time monitoring dashboard built to display key macOS system and network performance metrics in a clean, easy-to-read interface. This project was designed to improve visibility into workstation health, resource usage, and overall system behavior.

## Overview

The MacOS System & Network Dashboard provides a centralized view of important performance data such as CPU usage, memory consumption, disk utilization, network activity, and other core operating system statistics. It is intended to support system monitoring, troubleshooting, and performance awareness in a practical and visual way.

## Features

- Real-time system performance monitoring
- CPU usage tracking
- Memory and RAM utilization display
- Disk usage monitoring
- Network activity visibility
- Clean dashboard-style interface
- Designed for macOS environments

## Purpose

This project demonstrates how dashboard development can support system engineering by improving observability, simplifying performance analysis, and creating a more efficient way to monitor endpoint health. It reflects the importance of integrating software, system data, and user-focused design into one operational solution.

## Technologies Used

- Python
- HTML
- CSS
- JavaScript

## Project Structure

- `app.py` or main Python file for backend logic
- HTML for dashboard layout
- CSS for styling and presentation
- JavaScript for front-end interactivity
- Supporting files for metric collection and updates

## Use Case

This dashboard can be used by students, IT support teams, and system administrators who want a lightweight way to visualize and monitor macOS system behavior in real time.

## Outcome

The completed dashboard highlights the value of combining scripting, system monitoring, and interface design into a practical engineering solution. It serves as both a technical project and a demonstration of system engineering principles applied to endpoint monitoring.
